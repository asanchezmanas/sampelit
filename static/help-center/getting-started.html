Getting Started with Samplit

Welcome to the Samplit A/B Testing Platform. This guide will help you set up and run your first data-driven experiments in minutes.

schedule Last updated Dec 26, 2025
timer 8 min read
v1.0 Compatible

What is Samplit?

Samplit is an intelligent A/B testing platform that helps you optimize your website through data-driven experimentation. Unlike traditional A/B testing tools, Samplit uses Adaptive Optimization algorithms that automatically allocate more traffic to better-performing variations, maximizing your conversions while the test runs.

Key Benefits

Feature Benefit
Adaptive Algorithms Automatically send more traffic to winners
Visual Editor Create tests without coding
Multi-Element Testing Test multiple elements simultaneously
Real-time Analytics See results as they happen
Bayesian Statistics Faster, more accurate decisions

Key Concepts

Before you begin, understand these core concepts that drive the Samplit engine:

science

Experiments

A test comparing different versions of content with defined goals and traffic allocation.

layers

Variations

Different versions being tested: the Control (original) and your modified Variants.

Quick Start (5 Minutes)

1. Add Your Website

Paste the tracker snippet in your website's <head> section. This script is lightweight and executes asynchronously.

Installation Snippet 
<!-- Samplit A/B Testing Tracker -->
<script>
(function() {
    window.SAMPLIT_CONFIG = {
        installationToken: 'YOUR_TOKEN',
        apiEndpoint: 'https://api.samplit.com/api/v1/tracker'
    };
})();
</script>
<script src="https://api.samplit.com/static/tracker/t.js" async></script>

Your First Experiment

  1. Define Your Goal: Identify a clear KPI, like "Sign Ups" or "Add to Cart".
  2. Launch Visual Editor: Click "Create Experiment", enter your URL, and use our point-and-click editor to make changes.
  3. Set Variations: Create at least one variant (e.g., changing the CTA text from "Submit" to "Get Started Now").
  4. Start: Review and launch. Samplit will begin managing traffic automatically.

Understanding Results

Samplit uses Bayesian statistics to provide more accurate results faster than traditional testing.

  • Confidence: The probability that a variant is actually better than the control. Look for >95%.
  • Improvement: The percentage lift in conversion rate compared to the control.
  • Estimated Winner: Our AI engine's prediction of the winning variant based on current performance data.
