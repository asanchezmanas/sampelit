Visual Editor Guide

Create powerful experiments without writing a single line of code. Learn how to use our point-and-click editor to modify your website visually.

schedule Last updated Dec 26, 2025
timer 12 min read
Visual Builder v2.0

Overview

The Visual Editor is a point-and-click tool that lets you create A/B test variations without writing code. Simply load your website, click on elements, and modify them visually.

check_circle Point-and-click selection
check_circle Real-time live preview
check_circle Multiple element testing
check_circle CSS/HTML deep editing

Accessing the Visual Editor

You can access the editor through two main paths in the dashboard:

  • Click Visual Editor in the primary sidebar navigation.
  • While creating a new experiment: Experiments → Create New → Use Visual Editor.

Simply enter the full URL of the page you wish to optimize (e.g., https://yoursite.com/landing-page) and click Load Page.

Selecting Elements

Simply move your mouse over the page preview. Elements will highlight with a blue dashed outline as you hover. Click to select and lock an element for editing.

info

Selector Logic

Samplit automatically generates a robust CSS selector (e.g., button.cta-primary) to ensure your changes apply correctly even if the underlying HTML structure shifts slightly.

Creating Variations

After selecting an element, click + Add Variation. By default, variations are named Variant A, B, etc. We recommend renaming them (e.g., "Orange CTA") for better tracking in analytics.

Editing Content

Basic Edits

  • Text: Directly edit headings, paragraph text, or button labels in the sidebar.
  • Styling: Change colors, font sizes, padding, and borders using the intuitive sliders and pickers.
  • Images: Swap hero images or icons by uploading new files or providing a URL.
  • Visibility: Hide distracting elements with a single click to test a cleaner layout.

Advanced Editing

For designers and developers who need more control, the Visual Editor provides two powerful tabs:

Custom CSS

Custom HTML

Completely replace an existing element's structure by injecting custom HTML. This is ideal for adding icons to buttons or testing complex new components.

Troubleshooting

Issue: Page Not Loading

If your site fails to load, it is likely due to X-Frame-Options: DENY headers. Contact your developer to allow Samplit's domain as a frame ancestor, or use our Proxy Middleware for seamless integration.
