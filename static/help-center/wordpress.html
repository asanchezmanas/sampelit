WordPress Integration Guide

Turn your WordPress site into a conversion machine. Seamlessly connect Samplit to track engagement, test landing pages, and optimize WooCommerce stores.

schedule Last updated Dec 26, 2025
timer 18 min read
WP Plugin v2.5

Overview

The Samplit WordPress integration allows you to run A/B tests on your WordPress website with automatic tracking and real-time webhooks. This integration supports both WordPress.com hosted sites and self-hosted WordPress installations via our official plugin.

check_circle Gutenberg Block Editor sync
check_circle WooCommerce tracking add-on
check_circle OAuth 2.0 secure connection
check_circle Theme-agnostic integration

Integration Options

Choose the integration method that matches your WordPress setup:

Setup Method Difficulty
WordPress.com OAuth Connect Easy (One-click)
Self-Hosted Official Plugin Medium (Plugin install)
Custom / Headless Manual Script Advanced

WordPress.com OAuth

  1. In Samplit: Integrations → Add WordPress.
  2. Select WordPress.com.
  3. Authorize "Samplit Optimization Engine" on the WordPress.com permissions screen.
  4. Select the specific site from your network.

Self-Hosted WordPress

We recommend using our official plugin for the most robust tracking features, including full support for WooCommerce and Gutenberg blocks.

Option A: Official Plugin

  1. Download the samplit-ab-testing.zip from your dashboard.
  2. In WP Admin: Plugins → Add New → Upload.
  3. Activate and navigate to Settings → Samplit.
  4. Enter your Installation Token and save.

Option B: Manual child-theme integration

// Add to functions.php
add_action('wp_head', function() {
    echo '<script src="https://api.samplit.com/t.js" async></script>';
});

WooCommerce Support

Enable the WooCommerce add-on in plugin settings to automatically track:

  • Add-to-cart conversion rates.
  • Cart abandonment trends.
  • Total order value (Revenue) attributed to specific test variants.
  • Coupon usage in experiments.

Troubleshooting

info

Caching Plugin Warning

If you use WP Rocket, W3 Total Cache, or Autoptimize, ensure you clear your cache after installing Samplit. We recommend excluding the Samplit cookie from page caching for real-time accuracy.
